# Portfolio Website Specification

Build a modern, futuristic portfolio website for a Full-Stack JavaScript Developer named Jordan Waldehz.

## Tech stack

- HTML, Tailwind CSS, and vanilla JavaScript (no frameworks).

## Design style

- Dark theme (deep black/dark navy background)
- Neon accent colors (cyan #00f5ff or purple #7c3aed)
- Glassmorphism cards (backdrop blur, semi-transparent)
- Subtle animated particle or grid background
- Smooth scroll animations (fade-in on scroll)
- Clean, minimal, developer-aesthetic typography (use Inter or Space Grotesk font)

## Sections

1. **Hero** — Name, one-liner headline, CTA buttons (View Work, Contact Me), animated typing effect for role titles
2. **About** — Short bio, profile photo placeholder, tech stack icons
3. **Skills** — Animated skill bars or glowing badge pills for: React, Next.js, Node.js, Express.js, MongoDB, Tailwind CSS, JavaScript, HTML, CSS, Git
4. **Projects** — 3 glassmorphism cards with project name, description, tech tags, GitHub and Live Demo links
5. **Experience** — Timeline layout for Wyn City (Apr 2022 – Present)
6. **Contact** — Email link, GitHub link, LinkedIn link, simple contact form
7. **Footer** — Minimal with copyright

## Extra details

- Fully responsive (mobile-first)
- Sticky navbar with blur background on scroll
- Neon glow hover effects on buttons and cards
- Cursor glow effect (optional)
- All animations should be CSS or lightweight JS only

## Overall feel

High-end developer portfolio — think Linear.app meets a cyberpunk aesthetic.
